Installation in OsmAnd

    Android/data/net.osmand/files/tiles/ kopieren.
  2. OsmAnd öffnen → Karte konfigurieren → Kartenquelle → gewünschte Karte wählen.
  3. Optional als Overlay/Underlay unter Kartenebenen einblendbar.
