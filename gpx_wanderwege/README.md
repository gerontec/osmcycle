# Wanderwege GPX

Die drei Fernwanderwege („Höhenwege") aus der OSMCycle-App als GPX-Tracks,
extrahiert aus `app/wanderwege.json` (generiert mit `scripts/mk_gpx.py`).
Zum Import in OsmAnd: *Meine Orte → Tracks* — Farben sind via
`<osmand:color>` passend zur App hinterlegt.

| Datei | Weg | Farbe |
|-------|-----|-------|
| maximilian.gpx | Maximiliansweg (E4) | 🔵 blau #2b7ab5 |
| tirol.gpx | Tiroler Höhenweg | 🟢 grün #1a9640 |
| karnisch.gpx | Karnischer Höhenweg | 🔴 rot #d6191c |
